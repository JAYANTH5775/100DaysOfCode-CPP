# 100DaysOfCode-CPP

This repository contains daily C++ practice programs.

## Completed days
- Day 1: Number to words (0 to 100)
- Day 2: Palindrome number check
- Day 3: Star pattern
- Day 4: Subset sum (brute force using bitmask)
- Day 5: Factorial
- Day 6: Prime check
- Day 7: Fibonacci series
- Day 8: Array min and max
- Day 9: Linear search
- Day 10: Bubble sort

## Compile and run
Example:

```bash
g++ -std=c++17 Day-6PrimeCheck.cpp -o day6
./day6
```
